Protein Information Search

Use this search tool to find detailed information about proteins, including biological functions, molecular structures, and Gene Ontology terms.

Search Function Introduction

Here you can search for protein information by entering the protein ID or sequence.

The results will include details on the primary, secondary, and tertiary structures, as well as functional annotations.

This search supports proteins from Escherichia coli and Saccharomyces cerevisiae.
