Protein Interaction Search

Search Function Introduction

Here you can search for interactions for a given protein in a specific species, getting interaction details and an interaction network visualization graph.

The interaction details include two interacting proteins, their interaction score, the publications reporting the interaction, and the experimental methods used to discover the interaction.

The protein ID and species should be given, as well as the filter score which represents the lowest interaction score you want to get on this page.

As an alternative parameter, the filter score has a default value of 0, where all of the interaction proteins will be shown.
